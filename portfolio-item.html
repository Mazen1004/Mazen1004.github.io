Fotomatic Photography Website

Web Development Project

In the codecademy course for building a website, I was tasked with building a photography website for a virtual photography store. I used HTML and CSS to design the website and considered different aspects such as responsive design and the Box Model while implementing my solution.

The image above shows the main body of the webpage, CSS was used to adjust the image padding and margins to display the filters and images in an artistic manner.

The websites content flow and padding adjusts for different devices allowing for a smooth user experience, regardless of what device is used to view the webpage.
